## ⚡ QUICK START (5 MINUTES)

### 1️⃣ LOCAL DEVELOPMENT
```bash
npm install
npm start
# Visit http://localhost:3000
```

### 2️⃣ BUILD DOCKER IMAGE
```bash
docker build -t my-portfolio:latest .
docker run -p 80:80 my-portfolio:latest
# Visit http://localhost
```

### 3️⃣ DEPLOY TO KUBERNETES
```bash
# Step 1: Push image to registry
docker push your-registry/my-portfolio:latest

# Step 2: Update k8s/deployment.yaml
sed -i 's|IMAGE|your-registry/my-portfolio:latest|g' k8s/deployment.yaml

# Step 3: Deploy
kubectl apply -f k8s/deployment.yaml

# Step 4: Get URL
kubectl get svc web-service
```

## 🛠️ USEFUL COMMANDS

### Development
npm install               # Install dependencies
npm start                # Start dev server
npm run build            # Build for production
npm test                 # Run tests

### Docker
docker build -t img:v1 .        # Build image
docker run -p 80:80 img:v1      # Run container
docker push registry/img:v1     # Push to registry
docker logs <container-id>      # View logs

### Kubernetes
kubectl apply -f k8s/deployment.yaml    # Deploy
kubectl get svc web-service             # Get URL
kubectl logs -l app=web -f              # View logs
kubectl scale deployment web-deploy --replicas=5  # Scale
kubectl describe pod <pod-name>         # Pod info
kubectl port-forward svc/web-service 8080:80  # Access

## 📈 MONITORING

Check Application Health:
```bash

# Resource usage
kubectl top pods -l app=web

# Auto-scaler status

# Recent events
kubectl get events --sort-by='.lastTimestamp'
```
